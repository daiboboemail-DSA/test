# AI Studio 导出指南

## 如何向 AI Studio 提要求导出应用

### 1. 导出当前前端应用

**要求模板：**
```
请帮我导出当前设计的前端应用，要求：
1. 导出为 React + Vite 项目
2. 使用 Tailwind CSS 进行样式
3. 保留所有交互功能和动画效果
4. 导出完整的组件代码和样式
5. 确保代码可以直接运行（npm install && npm run dev）
```

### 2. 导出设计稿和资源

**要求模板：**
```
请导出设计资源：
1. 所有图片、图标资源
2. 颜色方案和设计规范
3. 字体配置
4. 组件设计规范文档
```

### 3. 导出完整项目结构

**要求模板：**
```
请导出完整的项目结构，包括：
1. 所有 React 组件文件
2. 样式文件（CSS/Tailwind）
3. 配置文件（package.json, vite.config.js 等）
4. 资源文件（图片、字体等）
5. README 说明文档
```

## 向 AI Studio 提要求的完整模板

```
我现在有一个案例库管理系统，前端展示页面已经完成。
现在需要你帮我设计一个管理后台，具体要求如下：

### 功能需求：
1. 案例管理
   - 案例列表（支持搜索、筛选、排序）
   - 案例详情查看
   - 案例编辑（标题、描述、标签、图片、视频）
   - 案例删除（支持批量删除）
   - 案例状态管理（发布/草稿/隐藏）

2. 标签管理
   - 标签列表
   - 创建/编辑/删除标签
   - 标签颜色设置
   - 标签使用统计

3. 媒体管理
   - 图片上传（支持多图）
   - 视频上传
   - 媒体库浏览
   - 媒体删除和替换

4. 数据统计
   - 案例总数
   - 按标签分类统计
   - 上传趋势图表

### 设计要求：
- 使用与前端一致的设计风格
- 响应式设计（支持桌面和移动端）
- 清晰的导航和操作流程
- 友好的用户交互体验

### 技术栈：
- React + Vite
- Tailwind CSS
- Supabase（数据库和存储）

请帮我设计这个管理后台的界面和交互流程。
```
